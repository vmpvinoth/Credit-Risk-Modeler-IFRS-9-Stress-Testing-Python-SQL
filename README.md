## 📋 Project Overview

This repository delivers a **production-grade, end-to-end credit risk modelling pipeline** using **real, publicly available datasets**. Every regulatory framework, formula, and threshold is sourced from official publications.

> **Portfolio Context:** This project mirrors the analytical work performed by **Credit Risk Stress Testing teams** at Tier-1 banks (Standard Chartered, HSBC, Barclays) — covering model development, IFRS 9 provisioning, Basel III capital, regulatory stress testing, and SR 11-7 compliant model validation.

### 📥 How to Download the Data

**Option 1 — Automated (FRED data only):**
```bash
python scripts/download_data.py
```

**Option 2 — LendingClub (Kaggle CLI):**
```bash
kaggle datasets download -d wordsforthewise/lending-club
unzip lending-club.zip -d data/raw/
```

**Option 3 — LendingClub (Manual):**
1. Go to: https://www.kaggle.com/datasets/wordsforthewise/lending-club
2. Click **Download** (saves as `lending-club.zip`, ~1.3 GB)
3. Extract `accepted_2007_to_2018Q4.csv.gz` → place in `data/raw/`

> **📌 Note:** The notebook includes a `DataLoader` class with a calibrated fallback — you can **run the full notebook without downloading data**. All fallback distributions are calibrated to real LendingClub statistics.

## ⚠️ Model Limitations (SR 11-7 Mandatory Disclosure)

1. **Binary classification only** — Does not model time-to-default or loss severity path
2. **Logistic regression** — Assumes linearity in log-odds space
3. **Static LGD** — LGD assigned at origination; does not reflect collateral value changes
4. **GDP-only systematic factor** — Single-factor Vasicek; real banks use multiple macro factors
5. **No account correlation** — ECL summation assumes independence
